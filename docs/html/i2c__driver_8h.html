Detailed Description

I2C driver for interfacing with ftd2xx devices.

Provides functions for initializing, writing to, reading from, and closing I2C channels of FTDI devices

All functions return an'FT_STATUS' value from the FTD2xx library. Handles of type 'FT_HANDLE' are used to reference the open I2C channel.

Note
Ensure the correct channel is initialized and passed to the write/read functions.
