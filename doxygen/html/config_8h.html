Detailed Description

Configuration definitions for AmPLink FTDI channels and GPIO mapping.

This header file provides enumerations and macro definitions to configure the FT4232H within the AmPLink device. It includes:

  • Channel identifiers for FTDI interfaces
  • SPI chip select bitmasks for libMPSSE usage
  • GPIO bitmask definitions for FTDI channels 2 & 3

These definitions are used throughout the AmPLink drivers. Use these constants within the functions defined in:

  • spi_driver.h
  • gpio_driver.h
Note
Ensure the correct FTDI channels are initialized before using macros or enums.
