Detailed Description

Handles AmpLink to Processor board specific protocols.

Provides the main interface for programming the Processor board through the AmpLink. Provides functions for all needed flash/clock commands and abstracts AmpLink away from the user Also contains some passthroughs for direct SPI/I2C communication
